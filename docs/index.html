Update your DeskTicker

This flashes the latest firmware straight from your browser — no Arduino IDE, no drivers to install. It takes about a minute.

  1. Use Google Chrome or Microsoft Edge on a computer (this does not work on phones, Firefox or Safari).
  2. Plug the DeskTicker into your computer with the USB-C cable.
  3. Put it in update mode: press and HOLD BOOT, briefly tap RST, then release BOOT. (The two small buttons on the back of the device.)
  4. Click Install below and pick the new serial port that appears.
  5. To keep your settings (Wi-Fi, tickers, theme), leave the “Erase device” box unchecked. Tick it only for a clean reset / recovery.
  6. When it finishes, press RST once to start the new firmware.
⚠ Your browser can’t flash devices. Please use Chrome or Edge on a desktop. ⚠ This page must be opened over https (or localhost).

Having trouble?

  1. No port appears, or it fails to connect: the device wasn’t in update mode. Redo step 3 (HOLD BOOT → tap RST → release BOOT), then click Install again.
  2. “Erase device” warning: erasing wipes your Wi-Fi and settings — you’ll redo the quick on-screen setup afterward. Skip it for a normal update.
  3. Wrong browser: only Chrome and Edge on a desktop/laptop can flash over USB.
  4. Need a clean start without a computer? On the device, triple-tap the screen → Settings → “Re-do Wi-Fi Setup”. (Or, as a hardware fallback, hold BOOT for 6 seconds.)
